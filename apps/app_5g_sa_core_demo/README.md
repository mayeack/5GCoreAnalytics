# app_5g_sa_core_demo - 5G SA Core Analytics

Executive demo app for 5G Standalone Core analytics in Splunk.

## Dashboards

1. **Service Health Overview** - Executive-level view of 5G SA Core health, subscriber impact, and active issues
2. **Operations Drilldown** - Operational investigation view for correlating alerts, NF health, and KPIs to isolate root cause

## Prerequisites

- `TA_5g_sa_core` must be installed with sample data loaded into the `fiveg_sa_core` index
